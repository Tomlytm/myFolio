Fashion App

Project information

  • Category: Mobile App
  • Client: Personal Project
  • Project date: 2024
  • Project Technologies: React Native

Fashion App is a modern mobile application designed for fashion enthusiasts. The app features a sleek and intuitive interface that allows users to browse through the latest fashion trends, discover new styles, and stay updated with fashion collections. Built with React Native, the app provides a seamless user experience across different platforms with smooth navigation and responsive design.
